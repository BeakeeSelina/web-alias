# 注册登录网站

一个简单的用户注册和登录系统。

## 功能特性

- 用户注册
- 用户登录
- JWT 身份验证
- 安全的密码加密

## 安装

```bash
npm run install-all
```

## 运行

开发模式（同时运行前端和后端）：
```bash
npm run dev
```

或者分别运行：

后端：
```bash
npm run server
```

前端：
```bash
npm run client
```

## 技术栈

- 前端：React + Vite
- 后端：Node.js + Express
- 数据库：SQLite
- 认证：JWT
